# **StreakMaster: Competitive Habit Tracking**

*A dynamic web application for building habits through streaks, challenges, and friendly competition.*

## **ğŸ“Œ Overview**

**StreakMaster** is an advanced habit-tracking platform designed to help users build long-term habits using streaks, progress analytics, and gamified competition features.

This project was developed following the **Agile SDLC**, enabling incremental delivery, continuous improvement, and iterative refinement of features.

## **âš™ï¸ Features**

### **1. User Account Management**

* Register, login, logout
* Update profile details

### **2. Habit Tracking Module**

* Create new habits
* Edit habit details
* Delete or archive habits
* Track daily habit progress
* Automatic calculation of streaks

### **3. Streak System**

* Continuous daily streak monitoring
* Display habit streak visually
* Streak reset when habits are missed

### **4. Analytics & Dashboard**

* Visual graphs of habit consistency
* Daily, weekly, and monthly habit reports
* Summary of highest streaks

### **5. Competitive Features**

* Join habit challenges
* Compare streaks with friends
* Leaderboards and competition rules

### **6. Notifications**

* Reminders for pending habits
* Alerts on streak breaks
